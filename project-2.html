Detalhes do Projeto

Este paramotor foi construído totalmente do zero. Fiz a estrutura, o depósito de combustível, o ressonador do escape, o sistema de arranque manual e outras peças. O motor era originalmente de uma scooter de 50 cc, que reconstruí e converti para 115 cc para garantir potência de voo fiável. Para a hélice, primeiro esculpi um modelo em madeira e depois construí uma pequena máquina para replicar hélices equilibradas. Assim consigo fabricar novas hélices sempre que preciso. Uso este paramotor regularmente para voar e estou sempre a fazer pequenas melhorias

Componentes Utilizados

Tubos de alumínio - estrutura soldada
Fio de kapron (corda de alta resistência) - para a rede de segurança
Acumulador artesanal - feito com baterias de telemóveis antigos
Motor de scooter de 50 cc convertido para 115 cc
Depósito de combustível personalizado - feito à mão
Ressonador do escape personalizado - afinado para melhor desempenho
Sistema de arranque manual - feito por mim
Máquina de replicação de hélices - para fabricar hélices equilibradas em madeira
Equipamento de soldadura - MIG/arc
Ferramentas de metalomecânica
Ferramentas básicas de eletrónica e cablagem
